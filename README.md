🚀 EduPresence Frontend (Next.js + React + TypeScript)

Frontend web do sistema de controle de presença acadêmica
**EduPresence**, desenvolvido com:

-   Next.js (React)
-   TypeScript
-   React Query
-   Axios
-   Tailwind CSS
-   Radix UI
-   Recharts
-   Jest + Testing Library

------------------------------------------------------------------------

# ⚙️ Configuração

## Instalar dependências

npm install

## Rodar em desenvolvimento

npm run dev

Aplicação disponível em: http://localhost:3000

------------------------------------------------------------------------

# 🔗 Integração com Backend

Defina no .env:

NEXT_PUBLIC_API_URL=http://localhost:3000

------------------------------------------------------------------------

# 🧪 Testes

npm test npm run test:watch npm run test:coverage
